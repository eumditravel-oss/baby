# 성남 산후조리원 모바일 비교 웹

GitHub Pages에 그대로 올릴 수 있는 정적 웹사이트입니다.

## 반영 내용

- iPhone 12 Pro 기준 앱형 UI로 재구성
- 대시보드 메뉴 추가
  - 조리원
  - 식단표
  - 체크리스트
  - 상담메모
- 조리원 메뉴를 첫 번째, 식단표를 두 번째로 배치
- 가로 스크롤 비교표 제거
- 모바일 카드형 랭킹 비교로 변경
- 하단 앱형 네비게이션 적용
- 상담 체크리스트 체크박스 기능 추가
- 카톡 요약 복사 기능 유지

## 배포 방법

1. 압축 해제
2. `baby-main` 폴더 안의 파일을 GitHub 저장소 루트에 업로드
3. GitHub Pages 활성화
4. `index.html` 접속
