# 성남 산후조리원 비교 앱

GitHub Pages에 그대로 배포할 수 있는 정적 모바일 웹앱입니다.

## 반영 내용

- iPhone 12 Pro 기준 앱형 UI 전면 개편
- 홈 화면 검색바와 추천순·거리순·가격순·평점순 필터 추가
- 홈 첫 화면에 추천 조리원 3개와 조리원/식단표 메뉴 우선 노출
- 조리원 목록을 세로 카드형 비교 UI로 재구성
- 가격, 거리, 평점, 마사지, 병원연계 등급 비교 카드 추가
- 식단표를 월별 카드형 메인 기능으로 변경
- 체크리스트를 분류별 체크 앱 형태로 변경하고 localStorage 저장 적용
- 상담메모를 조리원별 자동 저장 방식으로 변경
- 하단 네비게이션을 홈·조리원·식단표·체크·메모 5개 탭으로 확장

## 배포 방법

1. 저장소 루트의 `index.html`, `style.css`, `script.js`를 GitHub Pages에 배포합니다.
2. GitHub Pages를 활성화합니다.
3. 배포 URL 또는 `index.html`로 접속합니다.
