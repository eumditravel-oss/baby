# 성남 산후조리원 비교 웹

GitHub Pages에 바로 올릴 수 있는 정적 웹사이트입니다.

## 구성
- index.html
- style.css
- script.js

## 반영 내용
- 헤세드 산후조리원 명칭 반영
- 상단 비교표를 가격표/플랜표 스타일로 개선
- 상담 전 질문 체크리스트 추가
- 모바일 하단 빠른 이동 버튼 추가
- 카톡 요약 복사 버튼 추가

## 배포 방법
1. GitHub 저장소에 세 파일 업로드
2. Settings > Pages
3. Branch를 main, Folder를 root로 선택
4. Save

## 주의
가격, 마사지 비용, 차량 이동시간은 변동 가능성이 있으므로 실제 계약 전 조리원 상담으로 확인해야 합니다.
